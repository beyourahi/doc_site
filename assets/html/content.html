Accounts Manual

Chart Of Accounts

A.Highlight
  1. You do not need to input your Chart of Accounts in ERP. You provide your last year's accounts in Excel format & we can incorporate it into the system. You can just work on improvement/edit. ERP chart of accounts will never mismatch with you existing accounts.
  2. At the time of voucher entry all group of inventory codes will comes automatically under accounts code.
  3. There are 4-11 levels in code and within these level you can declare ledger head as per your necessity at any level
  4. No need to memorize code. At the time of voucher entry; you can search using alphabet
  5. Master Head & child heads all are independent so we can use all child head under one master head & one child head under multiple master heads.
  6. In the following example we declared the ledger head ‘Desktop’ at 4th level and in case of ‘Laptop’ at 11th level.
B. Chart of Accounts(Accounts Code) have 9 major groups:
  1. ASSETS
  2. Liabilities
  3. Income
  4. Direct Expenditure.
  5. Administrative Expense
  6. Brand Promotion/Marketing Expense
  7. Financial Expenses.
  8. Other Income.
  9. Non-Operating Expenses and Tax.

Accounts Opening

Opening Assets, Liabilities, Revenue and Cost Balances are inputted through ‘Accounts Opening’ Module:

To access Accounts Opening:
  1. Go to the Modules at the top left corner
  2. Click on the modules
  3. Look for Accounts & Finance Module below
  4. Press on the button
  5. Press on ‘Accounts Opening’ to enter.
User Interface
  1. Select the opening date
  2. Search for the account code
  4. Press on the button ‘2’ to access resource codes
  6. Enter Dr/Cr amounts (Quantity and amount) in the respective details code
  7. Press ‘Total’
  8. Press ‘Update’

Types Of Voucher

According to the Accounting Principle there are four types of voucher:

  1. Debit (Payment Voucher)
  2. Credit (Receipt Voucher)
  3. Contra Voucher
  4. Journal Voucher
To access Voucher:
  1. Go to the Modules at the top left corner
  2. Click on the modules
  3. Look for Accounts & Finance Module below
  4. Go to Operational Menu
  5. Press on Voucher Entry
  6. Select the required date
  7. Select the required type of voucher
  8. Select Control Code
  9. Press ‘OK’

(A) Debit (Payment) Voucher:

It is a voucher or document which is done against a vendor for doing the payment and can be used as a proof for monetary transaction between two parties. Also Payment voucher can be used for other purposes but generally it is done as a payment to vendor from the application. Payment voucher can also be done against a particular purchase order (PO) which is in due of that vendor.

In order to create a debit (payment) voucher.
  1. Select Date
  2. Type of Payment
  3. Select Control Code (Accounts Head)
  4. Press ‘Ok.’
  5. Select Head of A/C
  6. Sub of Account select (Resource Code)
  7. Specification select (if required)
  8. Bill no. (If against PO select Cost Center)
  9. Dr. Amount inputSelect Cheque date
  10. Input Cheque No
  11. Input Narration
  12. Press ‘Add’
  13. Recalculate (if edited)
  14. Save
  15. Print

Ledger

Ledger Book is a book of account that provides in depth record of financial transaction over the total lifetime period of a company or an organization. For a particular account there is always a ledger book for maintaining all the records.

To access Ledger Book:
  1. Go to the General Report
  2. Go to Ledger-All
  3. Select Ledger, Subsidiary Ledger Special Ledger (if needed)
  4. Select from – to date
  5. Select the Accounts Heads
  6. Press Show
  7. Press Print (if needed)
  8. Press on the Voucher no for details

MIS Report

To access MIS Report (Financal Position):
  1. Go to the MIS Module
  2. Select Statement of Financial Position
  3. Select From -To date
  4. Select Level (if needed)
  5. Press Ok
  6. Press Print (if needed)
To access MIS Report (Income Statement):
  1. Go to the MIS Module
  2. Select Statement of Comprehensive Income
  3. Select From -To date
  4. Select Level (if needed)
  5. Press Ok
  6. Press Print (if needed)
